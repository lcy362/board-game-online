极速101 是什么

极速101（又称 极速算力101）是一款卡通赛车主题的卡牌游戏类数学竞速桌游。你和对手轮流打出卡牌，把「速度」从 0 一路累加到接近 101；但只要你让速度超过 101，就会「爆胎」扣分。它把 100 以内的加减法、心算与博弈揉进一局几分钟的快节奏对局里，是家庭与聚会的轻量选择。

这一页为你准备了两样东西：① 免费的网页版，单人即可对战 1–5 名 AI；② 一份复刻实体桌游说明书的「规则百科」，整理出配件、目标、规则与功能牌一览，方便你随时查阅，也能直接回答「极速101 怎么玩」这类问题。

游戏目标

轮流出牌，把速度累加到当前总值；最后一名还握着轮胎的车手获胜。你要做的是：尽量把速度推高、逼近 101 拿优势，却在爆胎边缘精准收手。

怎么玩（核心规则）

  1. 准备：2–6 人（网页版为 1 名玩家 + 1–5 名 AI）。每人发 3 张手牌，拥有 3 个轮胎，速度初始为 0。
  2. 轮流出牌：轮到你时打出一张手牌，把牌值累加进「当前速度」；或打出功能牌触发特殊效果。
  3. 爆胎：一旦速度超过 101，立刻「爆胎」——失去 1 个轮胎、速度归零，并清除「升档加速」等临时增益。
  4. 淘汰：每名车手只有 3 个轮胎，轮胎用完即被淘汰出局。
  5. 胜负：等到只剩一名车手存活，对局结束并弹出结算，存活者夺冠。
小提醒：速度恰好等于 101 不爆胎，属于危险临界、安全过关；只有超过 101 才会爆。把速度精准停在 101 附近是高风险高回报的策略。

进阶技巧（怎么赢）

  • 留一张刹车保命：关键时刻降速，避免被迫爆胎。
  • 记牌：推测对手手里可能有什么，预判谁会被逼爆。
  • 用掉头 / 连续油门打乱节奏：给对手制造麻烦，尤其是逼玩家先爆。
  • 停在危险区：把速度压在 85–101，增加下家爆胎概率。
  • 提防困难 AI：它会主动顶高速度压你，务必预留余量。
