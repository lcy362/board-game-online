# 极速101（Speed 101）

> 单人 vs AI 的卡通赛车主题卡牌数学竞速游戏

---

## 一句话简介

你与 1–5 名 AI 车手同场竞速，用卡牌把「速度」一路累加到接近 101，却要在爆胎边缘精准收手——既比心算，也比博弈。

---

## 玩法规则

- **目标**：轮流出牌，把速度累加到当前总值；最后一个还握着轮胎的车手获胜。
- **爆胎**：一旦速度超过 101，立刻「爆胎」——失去 1 个轮胎、速度归零、清除升档状态。
- **淘汰**：每名车手只有 **3 个轮胎**，轮胎用完即被淘汰。
- **胜负**：等到只剩一名车手存活，对局结束并弹出结算。
- **卡组**：共 **74 张**
  - 速度牌：+1 ~ +12，每种 4 张（共 48 张），直接累加到当前速度。
  - 功能牌：9 种，共 26 张——
    - 刹车 −10、急刹 −20、加油 +18
    - 调速50（速度直接设为 50）
    - 超车（跳过下家，速度不变）
    - 掉头（反转出牌方向；2 人局相当于白赚一轮）
    - 连续油门（指定下家连出 2 张）
    - 升档加速（本回合可额外打出 1 张速度牌）
    - 失控反转（速度 = 101 − 当前速度）
- **对手**：AI 数量 **1–5**，难度 **简单 / 中等 / 困难**。
  - 简单：随机出牌，不主动进攻。
  - 中等：危险区（≥85）优先减速救急，会用升档、超车扩大优势。
  - 困难：在 ≥80 时主动把速度顶高逼你爆胎，更会配合升档打爆发。

---

## 操作说明

1. **开始屏**：选择 AI 数量（1–5）与难度（简单 / 中等 / 困难），点「发车」开局。
2. **出牌**：轮到你时，点击手牌即可出牌；非法牌（会让速度超过 101 却非必要的牌）会高亮提示。
3. **游戏内按钮**
   - 🌐 切换 **中文 / English**（即时生效）
   - ❓ 查看**完整规则**弹层
   - 🎓 重新观看**新手引导**
   - 功能牌卡面上的 **?** 可查看该单卡的效果说明
4. **出牌提示**：简单模式下会显示打出后的速度预测提示；**中等 / 困难模式不给出牌预测**，需要你自己心算收放手感。

## 语言支持

- **中文 / English** 双语，游戏内点 🌐 一键切换。
- 默认语言按**浏览器语言**自动选择（中文环境默认中文，其余默认英文）。

## 技术说明

- 纯**单文件 HTML5** + 原生 JS / CSS，**零依赖、零构建**。
- 浏览器直接打开 `index.html` 即玩，无需安装、无需联网、无需后端。
- 音效用 Web Audio 实时合成，不含任何外部资源文件。
- 适合托管到 **itch.io / GitHub Pages / 任意静态托管**：把含 `index.html` 的目录整包上传即可。
